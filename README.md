## Primeiros passos

- `yarn` ou `yarn install`

#### 1. Como rodar (dev)

- `yarn` ou `yarn install` (caso não tenha feito acima)
- `yarn dev`

#### 2. Build & Build Preview

- `yarn lint` ou `eslint ./src`
- `yarn build`
- `yarn preview` (opcional)

### Estilo Branches

Main + dev/feature, squash and merge após PR, deletar branch.
